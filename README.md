# 🚗 Autonomous Rover Navigation System

## Overview
This project simulates an autonomous rover capable of navigating environments with obstacle detection and adaptive pathfinding.

## Features
- Obstacle detection
- Dynamic path planning
- Sensor-based decision making

## Tech Stack
- C++

## How to Run
1. Compile:
   g++ src/rover.cpp -o rover
2. Run:
   ./rover
